How to Use the Recursion Visualizer

×

1. Function Name (REQUIRED)

Your recursive function must be named fn. The tracer only captures calls to fn.

def fn(n):        # Correct - will be traced
    ...

def solve(n):    # WRONG - will NOT be traced
    ...

2. Global Variables

Variables your function needs go in the Global Variables panel. Each row has a name and a value (any valid Python expression).

NameValue (Python expression)
arr[1, 2, 3]
target10
s'hello'
grid[[1,2],[3,4]]

Tip: Use single quotes 'text' for strings. Click + Add Variable to add more rows. Click × to remove one.

3. Function Call

In the input box at the bottom-left, type how to invoke your function:

fn(0, [])           # two arguments
fn(5)               # one argument
fn(0, 0)            # two integers
fn(0, set(), set()) # with sets

This must match your function's parameters exactly.

4. Indentation (IMPORTANT for Python)

Python requires consistent indentation. The editor uses 4 spaces per level:

  • Tab key → inserts 4 spaces
  • Shift+Tab → removes 4 leading spaces (dedent)
  • Enter → auto-indents to match the previous line
  • Format button → converts all tabs to 4 spaces and normalizes indentation
def fn(n):
    if n <= 0:           # 4 spaces (1 level)
        return 0         # 8 spaces (2 levels)
    return n + fn(n-1)   # 4 spaces (1 level)

⚠️ Never mix tabs and spaces. The Tab key and Format button always use spaces.

5. Custom Recursive Function Examples

Example A — Sum of digits:

Global variables: (none needed)
Function call:    fn(12345)

def fn(n):
    if n == 0:
        return 0
    return (n % 10) + fn(n // 10)

Example B — Array sum:

Global variables: arr = [3, 7, 1, 9]
Function call:    fn(0)

def fn(i):
    if i == len(arr):
        return 0
    return arr[i] + fn(i + 1)

Example C — Power function:

Global variables: (none needed)
Function call:    fn(2, 10)

def fn(base, exp):
    if exp == 0:
        return 1
    if exp % 2 == 0:
        half = fn(base, exp // 2)
        return half * half
    return base * fn(base, exp - 1)

Example D — Tower of Hanoi:

Global variables: (none needed)
Function call:    fn(3, 'A', 'C', 'B')

def fn(n, src, dst, aux):
    if n == 0:
        return 0
    left = fn(n-1, src, aux, dst)
    print(f"Move disk {n}: {src} -> {dst}")
    right = fn(n-1, aux, dst, src)
    return left + 1 + right

6. Tips & Limits

  • Keep total recursive calls small (≤ 80) for a clear tree
  • Use print() freely — output appears in the console panel (toggle it on)
  • Ctrl+Enter inside the editor runs the code
  • The speed slider controls animation speed (left = faster)
  • Only calls to fn(...) are traced — helper sub-functions won't appear in the tree
  • Use Export / Import links in the footer to save your work
